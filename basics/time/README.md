# Date & Time

Since Java 8, there's _Date & Time API_ (`java.time` package).

* Based on ISO-8601.
* Mostly immutable classes (__thread safe__).
* No public constructors (__static factories__)

## Commonly Used Classes

* `LocalDate`
* `LocalTime`
* `LocalDateTime`
* `ZonedDateTime`
* `Instant`
* `Duration` & `Period`
