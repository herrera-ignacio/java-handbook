# Access Modifiers

|               | Class | Package | Subclass<br/>(same pkg) | Subclass<br/>(diff pkg) | World  |

| `public`      |   +   |    +    |    +     |     +    |   +    | 
| `protected`   |   +   |    +    |    +     |     +    |        |

* \+ : accessible
